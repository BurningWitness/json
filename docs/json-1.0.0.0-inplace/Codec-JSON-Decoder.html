Description

Pure incremental RFC 8259 JSON decoder.

Correctness

All functions that parse or validate JSON object pair names treat surrogate Unicode code points as invalid input.

Laziness

Decoder functions that return concrete types in this library fully evaluate their results before returning unless their documentation specifies otherwise.
